üêõ Debug Information

If the extension isn't working, check:

  • Open Chrome DevTools (F12)
  • Go to the Console tab
  • Look for extension-related messages
  • Check if there are any error messages
  • Try opening chrome://extensions/ and reload the extension

Expected behavior:

  • Right-click menu should show "Get Stock Information for [selection]"
  • Popup should appear near the selected text
  • Popup should contain stock price, change, high/low, market cap
  • Popup should have links to Yahoo Finance and Google Finance
